Función nFibonacci(n)
    Si n es 0
        Retornar 0 // Caso base: el 0º número de Fibonacci es 0
    Sino Si n es 1
        Retornar 1 // Caso base: el 1º número de Fibonacci es 1
    Sino
        Retornar nFibonacci(n - 1) + nFibonacci(n - 2) // Caso recursivo: suma los dos números anteriores de la secuencia de Fibonacci (n-1 y n-2)
    Fin Si
