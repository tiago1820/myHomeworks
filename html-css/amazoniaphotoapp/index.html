The Amazon jungle includes territories belonging to nine nations

  • Brasil

  • Venezuela
  • Equador
  • Bolivia
  • Guiana
  • Suriname
  • Franca (Guiana Francesa)

There are three types of Amazon rainforest.

  1. Andean mountain forests
  2. Upland forests
  3. Flooded river forests
